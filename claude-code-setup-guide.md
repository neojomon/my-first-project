# Claude Code セットアップ手順メモ

> 作成日: 2026年3月27日
> 対象: 非エンジニア向け・後で見返すための記録

---

## Claude Code とは？

**Claude Code** は、Anthropic（アンソロピック）が作った AI アシスタント「Claude」をターミナル（黒い画面）から使えるようにしたツールです。
コードを書いたり、ファイルを編集したり、Git の操作を手伝ってもらったりできます。

---

## 事前に必要なもの

| 必要なもの | 説明 |
|------------|------|
| **Node.js** | JavaScript を動かすための仕組み（バージョン18以上） |
| **npm** | Node.js と一緒に入るパッケージ管理ツール |
| **Anthropic API キー** | Claude を使うための「鍵」。Anthropic のサイトで取得 |
| **Git** | ファイルの変更履歴を管理するツール |

---

## セットアップ手順

### ステップ 1：Claude Code をインストールする

ターミナルを開いて、以下のコマンドを入力・実行します。

```bash
npm install -g @anthropic-ai/claude-code
```

> **ポイント**
> `npm install -g` は「このパソコン全体で使えるようにインストールする」という命令です。
> `-g` は「グローバル（全体）」の意味。

---

### ステップ 2：API キーを設定する

Anthropic のダッシュボードで取得した API キーを環境変数に設定します。

```bash
export ANTHROPIC_API_KEY="ここにAPIキーを貼り付ける"
```

> **注意**
> API キーは他人に見せてはいけません。パスワードと同じです。
> ターミナルを閉じるとリセットされるので、毎回設定するか `.bashrc` / `.zshrc` に書いておくと便利です。

#### `.bashrc` に永続的に保存する方法（Mac/Linux）

```bash
echo 'export ANTHROPIC_API_KEY="ここにAPIキーを貼り付ける"' >> ~/.bashrc
source ~/.bashrc
```

---

### ステップ 3：インストールを確認する

```bash
claude --version
```

バージョン番号が表示されれば成功です。

---

### ステップ 4：プロジェクトフォルダに移動する

```bash
cd ~/my-first-project
```

> **ポイント**
> `cd` は「Change Directory（フォルダを移動する）」の意味です。
> `~` は「ホームフォルダ（自分のフォルダ）」を指します。

---

### ステップ 5：Claude Code を起動する

```bash
claude
```

これだけ！対話画面が起動したら成功です。

---

## よく使うコマンド一覧

| コマンド | 何をするか |
|----------|-----------|
| `claude` | Claude Code を起動する |
| `claude "質問や指示を書く"` | 一言で指示を出して結果だけ受け取る |
| `/help` | ヘルプを表示する |
| `/clear` | 会話の履歴をリセットする |
| `/exit` または `Ctrl + C` | Claude Code を終了する |

## GitHub との連携（今日やった作業）

### Git の基本的な流れ

```
編集する → ステージング → コミット → プッシュ
```

| 操作 | コマンド | 意味 |
|------|----------|------|
| 変更をステージング | `git add ファイル名` | 「このファイルを記録の対象にする」 |
| コミット（保存） | `git commit -m "メッセージ"` | 「変更内容に名前をつけて保存する」 |
| プッシュ（共有） | `git push -u origin ブランチ名` | 「GitHub にアップロードする」 |

### ブランチとは？

ブランチ＝「作業用の分岐」です。
本番のコードを壊さないよう、別の場所で作業するために使います。

```
main（本番）
  └── claude/my-feature（作業用ブランチ）← ここで作業する
```

## トラブルシューティング

### `claude: command not found` と出る場合

Node.js や npm が正しくインストールされていない可能性があります。

```bash
node --version   # バージョンが表示されるか確認
npm --version    # バージョンが表示されるか確認
```

表示されない場合は [Node.js 公式サイト](https://nodejs.org/) からインストールしてください。

---

### API キーのエラーが出る場合

- キーのコピペにスペースや改行が混じっていないか確認
- Anthropic ダッシュボードでキーが有効か確認
- `echo $ANTHROPIC_API_KEY` で設定されているか確認
